# 镜知 UI 设计升级说明

## 🎨 设计理念

**新中式科技美学** — 将传统东方意境与现代赛博朋克美学融合，创造独特的视觉体验。

### 核心主题：以镜观视，以知见意

设计灵感源于项目名"镜知"的哲学内涵，通过视觉语言传达：
- **镜** - 反射、观察、洞察的隐喻 → 流动的光影效果、镜面质感
- **知** - 智慧、认知、理解的体现 → 精准的信息层级、清晰的交互反馈

---

## ✨ 视觉特色

### 1. 色彩系统
- **主色调**：深邃墨色 `#0a0e1a` 
- **金箔光泽**：`#d4af37` → `#f4e4a6` 渐变（象征智慧之光）
- **青瓷蓝辉光**：`#2962ff` 点缀（科技感的东方表达）
- **中性灰**：`#8b95a8` 系列（优雅的信息层级）

### 2. 字体选择
- **中文**：Noto Sans SC（思源黑体）— 优雅现代的中文无衬线字体
- **英文/代码**：JetBrains Mono — 清晰的等宽字体，增强技术感
- **数字**：JetBrains Mono — 统一的代码风格

### 3. 材质与质感
- **毛玻璃效果**：`backdrop-filter: blur(20px) saturate(180%)`
- **多层阴影**：营造深度和悬浮感
- **内阴影高光**：`inset 0 1px 0 rgba(255, 255, 255, 0.05)` 模拟反光
- **边框光晕**：渐变边框 + box-shadow 创造辉光效果

## 🎯 设计亮点

### 1. 动态背景
```css
/* 三层叠加 */
- 径向渐变色彩场（蓝色/金色光晕）
- 精细网格纹理（80px × 80px）
- 浮动粒子背景（60s 循环动画）
```

### 2. 品牌标识动画
- **镜之符号 ◇** 持续脉冲动画（3s 循环）
- 缩放 + 旋转 + 辉光强度变化
- 金色到蓝色的双重阴影

### 3. 按钮交互
- **波纹扩散效果**：hover 时从中心向外扩散
- **光泽扫过动画**：渐变条从左到右移动
- **3D 悬浮感**：translateY + box-shadow 配合

### 4. 卡片设计
- **左侧彩色边框条**：hover 时渐显（金色→蓝色渐变）
- **图标悬浮反馈**：任务图标 hover 时放大 1.05x
- **状态标签辉光**：运行中状态带脉冲动画

### 5. 滚动条美化
- 自定义滚动条样式（金色 × 深色）
- `scrollbar-width: thin` + `::-webkit-scrollbar` 双重兼容

## 🔧 技术实现

### 关键 CSS 技术
1. **渐变文字**：`background-clip: text` 实现金色渐变文字
2. **动画流畅性**：`cubic-bezier(0.4, 0, 0.2, 1)` 缓动曲线
3. **背景混合**：多层 `radial-gradient` 叠加
4. **伪元素装饰**：`::before` / `::after` 创造装饰层
5. **过渡协调**：统一 0.3s-0.4s 的过渡时长

### 动画性能优化
```css
/* 使用 transform 而非 position */
transform: translateY(-4px);

/* 启用硬件加速 */
backdrop-filter: blur(20px);

/* 动画帧优化 */
will-change: transform, opacity;
```

## 📝 设计原则

1. **一致性** - 统一的金色调、圆角半径、间距系统
2. **层次感** - 通过阴影、模糊、透明度建立 Z 轴深度
3. **流畅性** - 所有动画使用协调的缓动曲线
4. **可读性** - 保持足够的对比度（WCAG AA 级别）
5. **文化融合** - 东方意境 × 赛博朋克 = 独特气质
